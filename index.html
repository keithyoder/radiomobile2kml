Create a RadioMobile coverage map using this guide.

The problem with using the png file that RadioMobile created in Google Maps is that it is impossible to create a transparent overlay. This utility will transform the png file into a kml polygon that can be used as a transparent overlay.

To use it, upload the kml and png files that were created by Radio Mobile. The form will return a kml file that can be opened in Google Earth or used as an overlay in Google Maps. I use this as a guide.
